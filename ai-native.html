Эволюция

Четыре волны AI в бизнесе

От инструментов к командам агентов. Каждая волна меняла отношение к AI.

2022
Инструменты
Человек спрашивает — AI отвечает. Одноразовое взаимодействие. Нет памяти, нет контекста, нет действий.
ChatGPT, Copilot, Midjourney
2023
Чат-боты
Все прикрутили GPT API к сайту. Отвечают на FAQ, генерируют текст. Но без памяти, без интеграций, без действий.
GPT-обёртки, Intercom AI, сценарные боты
2024
Агенты
AI-сотрудник. Помнит контекст, вызывает функции, использует инструменты. Действует, а не отвечает.
AI SDR, AI Support, Claude Code, function calling
2026
Agentic Workflow
Команда AI-агентов. Handoffs, shared knowledge, оркестрация. Не один агент — целый отдел.
Multi-agent orchestration, cofounder.run
Мы здесь. Agentic workflow — это сейчас.

Технологии

Почему сейчас возможно

Четыре технологических сдвига сделали AI-агентов реальностью.

🧠
LLM = мозг
Claude Sonnet, 200K контекст. Агент может удерживать всю историю клиента — сотни сообщений, документы, решения.
🐳
Контейнеры = тело
Docker-изоляция. Каждый агент — отдельная среда с файлами, памятью, инструментами. Безопасно и масштабируемо.
📉
Стоимость ↓
$3 за 1M токенов. Агент обрабатывает 1000+ диалогов в день за $10-30. Команда из 4-5 человек вместо 10 — при тех же объёмах.
🔧
Инструменты = руки
API-интеграции: CRM, календарь, email, мессенджеры. Агент может не просто говорить, а делать.
Мозг + Тело + Руки = AI-сотрудник
03 / 07
Тезис

Agentic Workflow

Принципиально другой подход. Не автоматизация диалогов — создание цифровых сотрудников.

Агент ≠ чат-бот
Агент = сотрудник с ролью, памятью, инструментами и KPI
✕ Чат-бот
  • ✕ Забывает после сессии — каждый раз с нуля
  • ✕ Только текст — не может записать в CRM или поставить напоминание
  • ✕ Отвечает шаблоном — не понимает контекст семьи
  • ✕ Ждёт вопроса — молчит, пока не напишут
  • ✕ Один сценарий — не передаёт контекст дальше
✓ AI-агент
  • ✓ Помнит всё — историю, предпочтения, причину отказа, бюджет
  • ✓ Записывает лида в CRM, бронирует тур, отправляет follow-up
  • ✓ Персонализирует — "Maria, мы подготовили встречу с преподавателем French programme"
  • ✓ Проактивный — напоминает за 24ч, 2ч, 30 мин до визита
  • ✓ Передаёт горячего лида live-менеджеру с готовым профилем
04 / 07
Технология

OpenClaw — AI-агент

Открытый AI-agent runtime. Каждый экземпляр — изолированный контейнер с личностью, памятью и инструментами.

👤
SOUL
Личность и правила агента. SOUL.md, IDENTITY.md. Определяет как агент общается, что делает и чего не делает. Полный контроль поведения.
💾
Память
Persistent filesystem. Агент помнит все разговоры, решения, контекст клиента. Между сессиями и перезапусками.
🔧
Инструменты
CRM, Calendar, Email, мессенджеры. Агент не просто отвечает — записывает в CRM, ставит напоминания, отправляет письма.
🔒
Изоляция
Docker-контейнер. Данные зашифрованы. Нет доступа к другим агентам. Безопасно как отдельный сервер.
🔑 Ed25519 идентификация
🧠 LLM через LiteLLM
⚡ Hot-config
🔄 Auto-recovery
