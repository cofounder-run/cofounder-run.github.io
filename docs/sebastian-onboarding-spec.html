Sebastian Onboarding Flow — ТЗ для Леонида

Обзор

Три момента: до Start → после Start → готово. Минимум сообщений, максимум ясности.

1. До Start (BotFather settings)

Что: описание бота + фото, видно ДО нажатия Start.

Bot photo: картинка краба Sebastian (файл: product/sebastian-crab.jpg — Маяк пришлёт)

Bot description (About):

🦀 Sebastian — your AI assistant with real hands.

I help you achieve your goals, save your time, and handle tasks independently.

🧠 Memory · ⏰ Automation · 🔍 Research · 🎤 Voice

Bot description (What can this bot do?):

Sebastian is a personal AI assistant that:
• Remembers everything between conversations
• Works on schedule — morning briefings, reminders, monitoring
• Researches, analyzes, and does tasks independently
• Understands voice messages

Press Start to set up your personal workspace →

Настройка: BotFather → /setdescription, /setabouttext, /setuserpic

2. После Start (платформа отправляет)

Когда: сразу после нажатия /start, ДО запуска контейнера или одновременно.

Одно сообщение (с картинкой краба):

🔧 Setting up your personal workspace — memory, goals tracking, automation.

This takes about 5 minutes. I'll message you when I'm ready!

Важно: - Это отправляет платформа (Rails), НЕ OpenClaw - Картинку краба отправить как фото с caption - Никаких промежуточных "loading...", "almost there..." — одно сообщение - Убрать текущие "Setting up your assistant...", "Machine started...", "Your assistant is ready! Send me a message"

3. Когда контейнер готов (OpenClaw / BOOTSTRAP.md)

Когда: контейнер запущен, OpenClaw работает — Sebastian САМ отправляет первое сообщение.

Сообщение:

✅ Ready! Let's get to know each other.

Feel free to send voice messages — I understand them perfectly, and it's faster than typing 🎤

What's your name?

Важно: - Sebastian НАЧИНАЕТ САМ — не "Send me a message to get started" - Это делает BOOTSTRAP.md (уже готов, файл product/sebastian-bootstrap.md / product/sebastian-bootstrap-en.md) - После этого сообщения идёт полный онбординг по BOOTSTRAP.md

Что убрать (текущие сообщения)

❌ "Setting up your assistant, please wait..." → заменить на п.2
❌ "Machine started, loading AI..." → убрать
❌ "Your assistant is ready! Send me a message to get started." → заменить на п.3 (BOOTSTRAP.md)
